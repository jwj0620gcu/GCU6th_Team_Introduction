# GCU 6th Team Introduction

가천대학교 GCS 6기 팀 소개 프로젝트입니다.  
전공자 2명(정원준, 전유정)이 비전공자(김채우, 이재빈)에게 VS Code 설치부터 바이브 코딩까지 함께 진행하며, 각자 랜딩 페이지를 제작하는 과정을 담았습니다.

![팀 단체 이미지](public/members/group.jpeg)

## 팀원 소개

### 정원준
- 학과: 소프트웨어학과
- 한 줄 소개: 뭐든지 열심히 하면 결국 빛을 본다.
- 관심 분야: 기획, 프론트엔드 개발
- 취미: 야구, 자전거, FC 서울 직관
- 키워드: 성공을 향해 꾸준히 나아가는 사람
### 김채우
- 학과: 경영학과
- 한 줄 소개: 가공된 껍데기를 PEEL하고, 선혈 낭자한 비즈니스의 본질을 '채우'하는 전략가
- 관심 분야: 마케팅
- 취미: 축구,러닝,헬스,ott시청
- 키워드: 매번이_반전, 파도같은_남자 

### 전유정
- 학과: 충남대 컴퓨터융합학부
- 한 줄 소개: 문제를 발견하면 그냥 지나치지 않고, 구조를 파악해 해결까지 설계하는 사람
- 관심 분야: 서비스 기획, 프로젝트 관리
- 취미: 뜨개질, 영화보기
- 키워드: 집돌이, 실행력

### 이재빈
- 학과: 기계공학과
- 한 줄 소개: 독립적이고 의연한 태도를 지닐 수 있는 자신만의 고유한 사고들을 가꿀 수 있게 끝없이 노력하는 사람
- 관심 분야: 역학, 철학, 진화학, 스타트업
- 취미: 운동, 독서, 음악 듣기
- 키워드: 시나브로, 고즈넉한



## 프로젝트 개요

- 목적: 팀원별 개성과 스타일을 살린 자기소개 랜딩 페이지 제작
- 진행 방식: 설치/환경세팅부터 코딩, 배포까지 실습 중심으로 진행
- 결과물: 팀 메인 랜딩 + 멤버별 상세 페이지 + 인터랙션 데모

## 랜딩 페이지 구성

- Home(메인): 팀 소개 진입 랜딩
- 원준 페이지: 포켓몬 스타일 인터랙션 자기소개
- 재빈 페이지: 절벽 앞에서도 끊임없지 나아가는 스스로 소개
- 채우 페이지: 하트 비트 형태 자기소개
- 유정 페이지: 불편함 강조 게임 스타일 자기소개

## 기술 스택

- Frontend: React, Vite, Tailwind CSS
- Routing: React Router
- Animation: CSS Animation, Framer Motion
- Deployment: Vercel

## 로컬 실행 방법

```bash
npm install
npm run dev
```
